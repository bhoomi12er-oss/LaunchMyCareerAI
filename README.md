# LaunchMyCareerAI 🚀

LaunchMyCareerAI is an AI-powered career development platform designed to help students and job seekers improve their career readiness through intelligent analysis and personalized guidance.

## Features

* 📄 AI Resume Analyzer
* 🎯 Skill Gap Analysis
* 🎤 Mock Interview Preparation
* 📊 Career Readiness Dashboard
* 🛣️ Personalized AI Career Roadmaps
* 🔐 User Authentication & Profile Management

## Tech Stack

### Frontend

* React.js
* Vite
* Tailwind CSS

### Backend

* Node.js
* Express.js
* MongoDB

### AI

* Google Gemini API

## Project Highlights

LaunchMyCareerAI helps users:

* Analyze resumes using AI
* Identify skill gaps for target roles
* Practice interview questions
* Track career readiness
* Receive personalized learning roadmaps
